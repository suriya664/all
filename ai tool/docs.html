Getting Started

Use your API key to authenticate and create your first AI job.

curl https://api.aitool.com/v1/jobs \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"type":"summarize","input":"Your text"}'
